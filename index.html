Introduction

Library Intelligent System(LiS) is the first system in the world where user only use face recognition to do the circulation(borrow n return book) in the library. User also will be introduced to Indoor Map where the function is to locate the book. New features in catalogue used to recommend user the latest books, recent books and others. This system also provide an automated intelligent cataloguing that will be used by the librarian to insert all the details for new book.

Face Recognition

Provide easy way to borrow and return book using face recognition

Indoor Map

The implementation of Indoor map to view current position of the books in the library

Automatic Intelligent Cataloguing

Facilitate information insertion using Optical Character Recognition (OCR) scanning of material images
